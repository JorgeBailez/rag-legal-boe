# Memoria del TFG

Carpeta para el documento de la **memoria del Trabajo de Fin de Grado** y materiales asociados
(figuras, anexos, versiones en PDF/DOCX). Se versiona en el repositorio para que el **tutor**
pueda seguir el avance directamente desde GitHub.

## Cómo se usa

- Añade aquí el documento de la memoria a medida que avances.
- Convención de nombres sugerida: `memoria_TFG_RAG_BOE_vN.pdf` (incrementa `N` en cada entrega
  significativa); conserva la última versión y, si quieres, un histórico.
- Las figuras/recursos pueden ir en `thesis/assets/`.

## Sobre el proyecto

Sistema **RAG informativo** sobre legislación consolidada del BOE. Aviso: los textos
consolidados del BOE tienen **carácter informativo** y no valor jurídico oficial; el sistema
**no** es asesoramiento jurídico. El código y los datos-evidencia del pipeline están en la raíz
del repositorio (`src/`, `scripts/`, `tests/`, `notebooks/`, `data/corpus/`, `data/manifests/`).
